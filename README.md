# 청소년기 가정 내 도서 보유 수와 성인기 문해력 분석 코드

이 저장소는 다음 논문의 분석 재현을 위한 R 코드를 제공한다.

**청소년기 가정 내 도서 보유 수가 성인기 문해력에 미치는 영향: PIAAC 2주기 한국 자료를 활용한 분석**

```text

```

## 2. 분석 자료

본 연구는 OECD 국제성인역량조사(PIAAC) 2주기 한국 Public Use File을 사용하였다.

원자료는 이 저장소에 포함하지 않는다. 분석을 재현하려면 OECD PIAAC 2주기 데이터베이스에서 한국 자료를 내려받은 뒤, 파일명을 다음과 같이 지정하여 이 저장소의 최상위 폴더에 두어야 한다.

```text
prgkorp2.csv
```

PIAAC 2주기 자료는 OECD PIAAC 2주기 데이터베이스에서 받을 수 있다.

```text
https://www.oecd.org/en/data/datasets/PIAAC-2nd-Cycle-Database.html
```

## 3. 분석 코드

주요 분석 코드는 다음 파일에 있다.

```text
book-ownership-and-adult-skills.R
```

이 코드는 다음 분석을 수행한다.

1. 14세 시점 도서 보유 수별 외생적 배경변수의 기술통계 및 교차분석
2. 14세 시점 도서 보유 수와 성인기 문해력의 복합표본 다중회귀분석
3. 미관측 교란요인에 대한 민감도 분석
4. 도서 보유 수와 부모 교육수준의 상호작용항 분석
5. 도서풍부 취약계층과 도서빈약 상위배경 집단의 사후 대비 분석

## 5. 실행 방법

먼저 OECD PIAAC 2주기 한국 자료를 내려받아 파일명을 다음과 같이 맞춘다.

```text
prgkorp2.csv
```

그다음 `prgkorp2.csv`와 `book-ownership-and-adult-skills.R`을 같은 폴더에 둔다.

R 또는 RStudio에서 다음 명령어를 실행한다.

```r
source("book-ownership-and-adult-skills.R")
```

분석이 끝나면 콘솔에 표 1부터 표 5까지의 결과가 출력된다.

## 6. 분석 원칙

본 분석은 다음 원칙을 따른다.

1. 결과변수는 문해력 plausible values인 `PVLIT1`부터 `PVLIT10`까지를 사용하였다.
2. 최종가중치 `SPFWT0`과 80개 반복가중치 `SPFWT1`부터 `SPFWT80`까지를 반영하였다.
3. 반복가중치에 기초한 분산 추정에는 Fay의 BRR을 적용하였으며, Fay 조정계수는 0.3으로 설정하였다.
4. 각 plausible value별 분석 결과는 Rubin의 결합규칙에 따라 통합하였다.
5. 분석에는 결과변수, 원인변수, 통제변수, 최종가중치 및 반복가중치에 결측이 없는 완전사례만 포함하였다.

## 7. 주요 변수

분석에 사용한 주요 변수는 다음과 같다.

```text
결과변수: PVLIT1-PVLIT10
원인변수: J2_Q06, 14세 시점 가정 내 도서 보유 수
최종가중치: SPFWT0
반복가중치: SPFWT1-SPFWT80
```

통제변수는 성별, 연령, 부모 교육수준, 14세 당시 부모의 경제활동 여부, 거주지역 규모, 가족구조이다.

## 8. 민감도 분석

민감도 분석은 Cinelli와 Hazlett(2020)의 접근을 바탕으로 수행하였다. 분석 대상 효과는 다중회귀분석에서 가장 큰 추정치를 보인 `201-500권` 집단의 효과이다.

민감도 분석에서는 기준 공변량으로 `부모 교육수준: 대졸 이상`을 사용하였으며, 가상의 미관측 교란요인이 이 기준 공변량의 1배, 2배, 3배 수준의 설명력을 갖는 경우를 비교하였다.
